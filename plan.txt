Структура библиотеки и спецификация использования

библитека должна работать как NPM package  - @sever-ui/react, возможно она будт расширена и для других фрейворков

Необходимо создать и настроить монорепозиторий с использованием Lerna либо Changesets

Библиотека должна работать на React + TS + Stitches

Для визуального тестирования использовать будем Storybook
Для тестов функционала будем использовать Jest

Форматирование и анализ
Prettier, Eslint

Во вторую очередь необходимо настроить Семантическое версионирование для правильной работы версий библиотеки

В третью настроить CI/CD

Компоненты библиотеки должны импортироваться из папки вот так
import {Button, Input, Theme, ...} from '@sever-ui/react';


Примерная структура библиотеки

ROOT

    dist - папка которая будет загружаться на NPM и доступна для загрузки как @sever-ui/react
        button - компонент
        input - компонент
        theme - компонент
        ...

    packages - корневая папка с библиотекой
        react - корневая папка с компонентами которая будет идти в dist
            button
            input
            theme
            ...
